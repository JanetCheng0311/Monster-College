Run locally

To run the full game locally with Python and Pygame:

python -m pip install -r requirements.txt
python Monster_College.py

If videos or images are missing, ensure the menu&map/intro_video and menu&map assets are present.

Notes & Publishing

- This static page links to the repository assets by relative paths. To serve the whole repo via GitHub Pages, set the Pages source to main branch (root) in the repository Settings.

- Alternative: place site files into a docs/ folder and publish from there, but ensure you copy required videos into docs/.
