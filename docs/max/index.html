Max Mini Game

This mini-game is currently a Python/Pygame application in the repository.

Run locally

  1. Install Python 3.10+ and create a virtualenv.
  2. Install dependencies: pip install pygame (and others if required).
  3. Run: python "Max_mini_game/Max Mini Game.py"
